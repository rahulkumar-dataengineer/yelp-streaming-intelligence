Yelp Streaming Intelligence

Real-time dual-pipeline · BigQuery + Qdrant · LangGraph

About this project

A real-time dual-pipeline platform that ingests 1M+ Yelp reviews through Redpanda, processes them with PySpark Structured Streaming (Bronze → Silver → Gold), and sinks simultaneously into BigQuery (SQL analytics) and Qdrant (768-dim semantic vectors). A LangGraph multi-agent router classifies each question and dispatches to the right pipeline — or both — for hybrid answers.

Architecture

Yelp JSON → Redpanda → PySpark Bronze
→ Silver (join) → Gold (dual-sink)
→ BigQuery | Qdrant
→ LangGraph Router → Gemini Synthesizer

Tech Stack

Streaming Redpanda (Kafka), PySpark
Structured Store BigQuery (free tier)
Vector Store Qdrant (self-hosted)
LLM / Embeddings Gemini Flash-Lite, gemini-embedding-001
Agent Orchestration LangGraph + LangChain
API Flask on GCP e2-micro
Frontend Firebase Hosting (static)

Why This Matters

Medallion Architecture at Scale

Bronze → Silver → Gold pipeline with stream-stream joins, watermarking, and dual-sink fanout — the pattern used in production data lakehouses.

Intelligent Query Routing

LangGraph flat graph with explicit SQL / VECTOR / HYBRID routing. Deterministic, debuggable, and cheap — no free-form ReAct loops for classification.

Zero-Cost Production Deployment

Runs entirely on GCP free tier: e2-micro VM, BigQuery Sandbox, Gemini free-tier APIs. Designed to demonstrate cost engineering alongside technical depth.

Enterprise Team Boundaries

Infra team (schema registry, Hive metastore) and DE team (pipeline) modeled as separate codebases with no cross-imports — mirrors real org structures.
