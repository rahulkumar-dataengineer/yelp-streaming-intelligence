Yelp Streaming Intelligence

About this project

A real-time streaming platform that builds the data infrastructure AI agents need. 1M+ Yelp reviews flow through Kafka into a PySpark medallion pipeline (Bronze → Silver → Gold), landing in two sinks: BigQuery for structured analytics and Qdrant for 768-dim semantic search. A LangGraph agent decides which sink answers each question — SQL, vector, or both.

Architecture

Yelp JSON → Kafka → PySpark Bronze
→ Silver (join) → Gold (dual-sink)
→ BigQuery | Qdrant
→ LangGraph Router → Gemini Synthesizer

Tech Stack

Streaming Kafka, PySpark
Structured Store BigQuery (free tier)
Vector Store Qdrant (self-hosted)
LLM / Embeddings Gemini Flash-Lite, gemini-embedding-001
Agent Orchestration LangGraph + LangChain
API Flask on GCP e2-micro
Frontend Firebase Hosting (static)

Why This Matters

Medallion Architecture at Scale

Bronze → Silver → Gold pipeline with stream-stream joins, watermarking, and dual-sink fanout — the pattern used in production data lakehouses.

Intelligent Query Routing

LangGraph flat graph with explicit SQL / VECTOR / HYBRID routing. Deterministic, debuggable, and cheap — no free-form ReAct loops for classification.

Dual-Sink Design with Vector Search

A single streaming job fans out to BigQuery (structured analytics) and Qdrant (semantic search) independently. 768-dim Gemini embeddings enable similarity search over 1M+ reviews — the AI layer queries both sinks, not just one.

Zero-Cost Production Deployment

Runs entirely on GCP free tier: e2-micro VM, BigQuery Sandbox, Gemini free-tier APIs. Engineering within constraints, not throwing money at problems.
